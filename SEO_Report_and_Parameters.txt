KALPNOVA WEBSITE - SEO REPORT & PARAMETERS
===================================================

---------------------------------------------------
1. PARAMETERS REQUIRED FOR HIGH SEO RANKING 
---------------------------------------------------
To achieve a high rank on Google, the following parameters are strictly required for this website moving forward:

1. CONTENT DEPTH (The most critical missing piece)
   - Search engines rank text, not visuals or 3D animations. 
   - Required: 800-1500 words of deeply informative, naturally written content on major pages (Home, About, Services).
   - Parameters: Include targeted keywords related to "Premium Web Design," "Brand Growth," and "Digital Agency Ahmedabad."

2. DYNAMIC PAGE TITLES & DESCRIPTIONS
   - Every single page must have a 100% unique "Title" and "Description" meta tag.
   - Required: You must provide exact marketing copy for: Home, About, Services, Products, Contact, and Hub pages.

3. PAGE PERFORMANCE (Lighthouse Core Web Vitals)
   - Because the site uses heavy 3D components, load speed can hurt rankings.
   - Required: Lazy-load offscreen images, compress 3D canvas textures, and ensure mobile loading is under 3 seconds.

4. ACCESSIBILITY (Alt Text)
   - Required: Every single image or visual graphic must have an 'alt' text parameter describing exactly what the image is (Search engines are blind, they can only read the alt text).

5. PAGE AUTHORITY (Backlinks)
   - Required: High-quality external websites (blogs, LinkedIn, directories) must link back to Kalpnova.com to prove authority.


---------------------------------------------------
2. CHANGES IMPLEMENTED SO FAR (For Non-Technical Users)
---------------------------------------------------
We have successfully built the "Technical SEO Foundation." Here is what has been coded into the website:

* CREATED A SITEMAP (sitemap.xml)
  What it does: Gives Google a literal "map" of every page on the website so it can find them all immediately without having to dig.

* ADDED CRAWLER INSTRUCTIONS (robots.txt)
  What it does: Gives search engines explicit permission to safely read the website.

* ADDED OPEN GRAPH & SOCIAL PREVIEWS
  What it does: When someone shares the website link on WhatsApp, LinkedIn, Twitter, or Discord, it will now display a nice image preview, bold title, and description instead of a blank gray box.

* INSTALLED REACT-HELMET-ASYNC
  What it does: This is the engine that allows Kalpnova to change the invisible Title and Description tags depending on which page the user clicks on.

* INJECTED SCHEMA MARKUP (JSON-LD)
  What it does: Adds hidden "Structured Data" code that explicitly tells Google: "We are an Agency, our name is Kalpnova, we are located in Ahmedabad." This helps trigger Rich Results in Google Search.

* SETUP BUILD-TIME PRE-RENDERING
  What it does: Because this website is built with React (a Single Page Application), it looks mostly blank to older search engines. We installed a tool that takes a "snapshot" of the website code so search engines can read it clearly like a simple blog.
